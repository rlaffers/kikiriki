NEXT STEPS:
-----------

1. Run the setup script:
   
   cd ~/bin/push-to-talk
   ./stt-setup

   This will:
   - Install required system packages (xbindkeys, xdotool, etc.)
   - Configure ~/.xbindkeysrc with default hotkey (Ctrl+Alt+G)
   - Set up auto-start on login
   
2. Start the service:

   ~/bin/push-to-talk/stt-start

3. Test it:
   
   - Press and HOLD Ctrl+Alt+G
   - Speak something
   - Release the keys
   - Your text will appear!


MANUAL TESTING (without xbindkeys):
------------------------------------

You can test the script manually:

   ~/bin/push-to-talk/speech-to-text-ptt start
   # speak something
   ~/bin/push-to-talk/speech-to-text-ptt stop


CUSTOMIZATION:
--------------

• Change keyboard shortcut:
  Edit ~/bin/push-to-talk/xbindkeysrc (see xbindkeysrc.example for ideas)
  Then restart: ~/bin/push-to-talk/stt-stop && ~/bin/push-to-talk/stt-start
  
• Change Whisper model, audio settings, etc:
  Edit ~/bin/push-to-talk/speech-to-text-ptt
  
• Change behavior (auto-paste vs clipboard):
  Edit AUTO_PASTE=true/false in speech-to-text-ptt

- Check microphone permissions
